Project Title : Student Club & Event Participation Tracker

Project Description :

The Student Club & Event Participation Tracker is a database-driven web application designed to manage university clubs, their members, and event activities efficiently.
It centralizes data related to clubs, events, participants, and achievements — helping students, club executives, and administrators maintain transparent records and track engagement statistics. The system allows clubs to register events, manage participants, calculate attendance, and generate detailed reports for performance evaluation and recognition purposes.

Unique Feature :

Reward Point System: Assigns points for each event attended; cumulative score used for certificates or ranking.

Feedback & Rating Module: Members can rate events and provide feedback for improvement.

Leaderboard Dashboard : Dynamic ranking of students or clubs based on cumulative participation points or feedback ratings.

Inactive Member Auto-Detection : SQL trigger or procedure detects members inactive for >3 months and marks them as “Inactive”.
